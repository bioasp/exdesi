Installation
------------


You can install exdesi by running::

	$ pip install --user exdesi

On Linux the executable scripts can then be found in ``~/.local/bin``



Usage
-----

Typical usage is::

	$ exdesi.py networkfiles expvar.txt  

For more options you can ask for help as follows::

	$ exdesi.py -h
